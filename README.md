# Dự Án Phân Tích & Dự Đoán Giá Cổ Phiếu 📈

Dự án này sử dụng ngôn ngữ Python để lấy dữ liệu chứng khoán thực tế và đưa ra các nhận định cơ bản dựa trên chỉ báo kỹ thuật.

## ✨ Tính năng
* Tự động lấy dữ liệu lịch sử mã cổ phiếu từ Yahoo Finance.
* Tính toán đường trung bình động (MA20) để xác định xu hướng.
* Vẽ biểu đồ trực quan giữa giá thực tế và đường xu hướng.
* Đưa ra khuyến nghị **MUA** hoặc **THẬN TRỌNG**.

## 🚀 Cách sử dụng
1. Copy mã nguồn trong file `analysis.py`.
2. Mở [Google Colab](https://colab.research.google.com/).
3. Cài đặt thư viện: `!pip install yfinance`.
4. Dán code và nhấn **Run**.
